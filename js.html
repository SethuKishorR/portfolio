These mini JavaScript projects highlight my expertise in creating interactive web applications, ranging from calculators to games. They showcase my coding prowess and passion for web development.

Text to Speech Converter : This website empowers users to input text or paste content into an input field, converting it into audio for easy listening. Users can control playback with options to play, stop, refresh or clear the input field, providing a seamless and interactive experience.

Link Manager : This website offers a robust solution for storing and managing links, ensuring data persistence even after system refresh or power-off. Users can effortlessly add and remove links, creating a seamless and reliable platform for organizing their online resources.

People Counter : Enter your email in this website, and watch the counter increase by +1 automatically upon submission, creating a seamless user-driven tally.

Data Hub: Users can securely input their email and mobile number on my website, with their data stored in a database for future reference. A dedicated 'Previous Entries' link leads to a separate page displaying a history of past user entries, enhancing transparency and accessibility.
